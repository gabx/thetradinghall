# TheTradingHall servers

This project is currently under heavy construction. You will find in the [Wiki pages](https://github.com/gabx/thetradinghall/wiki) a very detailed How-to about the server configurations. Some pages are still under writting and may contain few issues, some projects have been abandoned, some pages are not yet well formated.

All configurations and How-To have been tested and successfuly used. They result from lots of reading and are not only copy/paste recipies.

## Quick intro

On the first phase of the project, the server was hosted on Arch-linux home station and run with systemd container. After a period of test, everything went live with bare metal servers running Fedora Atomic.
